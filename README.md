# Computer Networks Laboratory Portfolio

## Overview
This repository serves as a centralized portfolio for my Computer Networks laboratory coursework. The primary objective of this repository is to document and version control my progression through the curriculum by pushing each laboratory assignment and experiment incrementally. 

## About the Computer Networks Laboratory
The Computer Networks laboratory provides practical, hands-on experience with fundamental networking concepts, protocols, and architectures. Throughout this course, the practical work typically encompasses:

*   **Network Topologies and Design:** Simulating and analyzing various network architectures using industry-standard tools.
*   **Protocol Analysis:** Utilizing packet sniffing tools to capture, inspect, and understand network traffic and communication protocols across the OSI model.
*   **Routing and Switching:** Configuring static and dynamic routing protocols (e.g., RIP, OSPF) and implementing VLANs on network switches.
*   **Subnetting and IP Addressing:** Applying IPv4 and IPv6 addressing schemes, calculating subnets, and managing IP address allocation within localized networks.
*   **Application Layer Programming:** Developing socket programs to understand client-server architectures and data transmission over TCP/UDP.
*   **Network Security Fundamentals:** Implementing basic security measures, such as access control lists (ACLs) and understanding network vulnerabilities.

## Repository Structure
Each subdirectory within this repository corresponds to a distinct laboratory session or experiment. As the coursework progresses, new directories will be added sequentially, containing all related configuration files, scripts, topology designs, and observation reports.
